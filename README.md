# HelvetiForma - Projets Combinés

Ce dépôt contient l'évolution complète de la plateforme HelvetiForma, de la version 1 à la version 3 révolutionnaire.

## 📁 Structure du Projet

### `helvetiforma/` - Version 1 (Originale)
Première version de la plateforme avec WordPress classique.

### `helvetiforma-tutor-lms/` - Version 2 (LMS)
Version intermédiaire avec intégration TutorLMS avancée.

### `helvetiforma_v3/` - Version 3 (Révolutionnaire) ⭐
**Version actuelle et recommandée** - Plateforme moderne Next.js 15 + React 19 avec système de contenu Markdown révolutionnaire.

## 🚀 HelvetiForma v3 - Innovation Majeure

### Technologies de Pointe
- **Next.js 15.5.4** + **React 19.1.0** + **TypeScript**
- **Tailwind CSS v4** avec configuration inline moderne
- **Framer Motion** pour animations fluides
- **Interface Admin "Notion-like"** révolutionnaire

### Innovation : Gestion de Contenu Simplifiée
❌ **Fini les ACF complexes et les interfaces difficiles**  
✅ **Fichiers Markdown + Interface admin intuitive = Simplicité maximale**

### Fonctionnalités Clés
- ✅ **Interface d'édition complète** avec prévisualisation temps réel
- ✅ **Création de pages** avec génération automatique de slug
- ✅ **Intégrations API** WordPress/TutorLMS + Microsoft Teams
- ✅ **Performance optimale** et prêt pour production

## 🎯 Démarrage Rapide (v3)

```bash
cd helvetiforma_v3
npm install
npm run dev
```

Accédez à :
- **Site principal** : http://localhost:3000
- **Interface admin** : http://localhost:3000/admin
- **Édition de contenu** : http://localhost:3000/admin/content/pages
